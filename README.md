# Zip's Blooket Hacks & Cheats GUI (Refactored)

An improved, modular, and aesthetically pleasing GUI for Blooket hacks. This project is a refactored version of the original "Zip's Blooket Hacks" designed for better performance and ease of use.

## Features

- **Modern UI**: Sleek dark theme with glassmorphism effects.
- **Draggable & Minimized**: Keep the GUI out of the way when you don't need it.
- **Search Functionality**: Quickly find the cheat you're looking for.
- **Anti-Cheat Protection**: Built-in blocker for Blooket's anti-cheat reports.
- **Modular Cheats**: Organized by game mode (Global, Gold Quest, Crypto Hack, Cafe, Factory, etc.).
- **Easy Keybind**: Press `Ctrl + E` to toggle the GUI visibility.

## How to Use

### Method 1: Bookmarklet (Recommended)

1. Copy the code below:
   ```javascript
   javascript:(function(){fetch('https://raw.githubusercontent.com/Zip-On/Zips-Blooket-Hacks-And-Cheats-GUI/main/Gui.js').then(r=>r.text()).then(eval)})()
   ```
2. Create a new bookmark in your browser.
3. Paste the code into the "URL" or "Address" field of the bookmark.
4. While playing Blooket, click the bookmark to open the GUI.

### Method 2: Console

1. Copy the entire content of `Gui.js`.
2. While playing Blooket, press `F12` to open the Developer Tools.
3. Go to the "Console" tab.
4. Paste the code and press `Enter`.
